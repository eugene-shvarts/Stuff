        • The starting value
        • The tolerance of the convergence criteria
        • Maximum number of iterations
        • A debugging option (e.g., to print status to the user)

    3. In this question, you will implement EM for a multivariate-t regression model:
      Yi∼tp(μ,Ψ,ν),i=1,…,n(3)
      where tp is a multivariate t-distribution in p dimensions. It turns out that the t-distribution has a convenient representation as a ratio of a normal and χ2 distribution. Therefore, we can write the model in (3) as:
      Yi|τi,θτi∼Np(μ,1τiΨ),∼χ2νν,(4)(5)
      where θ=(μ,Ψ) is the parameter to be estimated (ν is fixed) with Yobs=(Y1,…,Yn) and Ymis=(τ1,…,τn). For convenience, define:
      τ(t+1)i=E[τi|Yobs,θ(t)]=ν+pν+(Yi−μ(t))T[Ψ(t)]−1(Yi−μ(t)).(6)
      Note that there is no ⋅√ for the τi's in the multivariate formulation used here (unlike in the standard univariate representation of a t-distribution). There are multiple different definitions of the multivariate t−distribution in the literature, although the one used here is standard in the EM and Bayesian communities.
      1. Derive the EM algorithm for estimating θ i.e., find μ(t+1) and Ψ(t+1).

        2. Some possibly useful facts:

        • For A a p×p matrix and c>0 a constant:
          det(cA)=cpdet(A)(7)
        • If X∼Gamma(a,b) then:
          p(x|a,b)∝xa−1exp{−bx},(8)
          for a,b>0 and E[X]=ab.
        • If A is a positive definite p×p matrix then:
          argmaxA{nlog(det(A−1))−tr(A−1B)}=1nB.(9)
        • Facts about the trace operator for scalar c, vector x and square matrices A,B,C:
          tr(c)=c,tr(AB)=tr(BA)tr(A+B)=tr(A)+tr(B),tr(xTAx)=tr(xxTA).(10)(11)


    4. In this question you will implement two different EM algorithms for a Hierarchical Poisson model:
      Yi|λi∼Pois(λi),λi|β∼Gamma(1,β),(12)
      where the pdf of a Gamma(1,b) random variable is defined to be
      p(x)=I{x>0}bexp{−bx},(13)
      i.e., λi∼Expo(β).
      1. Derive the EM algorithm for β for the model in (12).
      2. Construct an Ancillary Augmentation (AA) that preserves the observed data log-likelihood of the Hierarchical model in (12). Hint: Use the strategy suggested in class! Derive the corresponding EM algorithm.
      3. Using the SA and AA from (a) and (b), derive the corresponding Interwoven EM algorithm. You may select either the SA or AA as A1 (the resulting will be the same either way for this example).
      4. Derive the observed data log-likelihood according to model (12) and compute the MLE for β.
      5. Compare the linear rate of convergence of each of the EM algorithms, and discuss when each algorithm will perform well (or not).
